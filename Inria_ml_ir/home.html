Error detection in MPI Aplications
using deep learning

The project is about building an A.I based program in python that allow us to detect and predict errors in multithread apps and parallel code. The subject is also about extending the MPI code database in order to train our detection model using shell scripts and procreation methods. The internship requires a good sense of data analysis, knowledge, and experience in Python, Shell, and MPI programming. It also requires good knowledge of Linux and other operating systems to test the program in different ways and an introduction level of feature engineering using LLVM-IR compiler.
